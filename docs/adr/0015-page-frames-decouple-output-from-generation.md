# Page Frames decouple output from generation

Historically one **Output Profile** did two jobs: its drawable aspect supplied the **Composition Frame** that generated a Scene, and its physical dimensions mapped that Scene onto the output page. We decided a committed **Page Frame** may decouple those jobs. The Page Frame is an axis-aligned final output boundary positioned in the original Composition Frame coordinates. In the ordinary, scale-preserving framing operation, moving or resizing that boundary crops or pads the page while the profile changes to preserve the Scene-to-physical scale. An explicit fixed-page inverse instead leaves the complete Output Profile—including all four physical insets—exactly unchanged and varies the Page Frame extent proportionally, which uniformly scales the frozen Scene behind the stationary physical Page. Neither operation regenerates the Sketch, mutates its Scene, or changes the frozen Composition Frame; recompose remains the explicit way to make the Sketch respond to a new page aspect.

This preserves exact generated artwork while making page framing reproducible. Once Page and Composition diverge, a **Preset** retains the original generation aspect, the Page Frame, and the final Output Profile; final preview and export clip and rebase completed geometry through that frame after expensive generation and Hidden-line work. Physical plot margins remain separate, and ADR-0009 background precedence remains unchanged across padded extent.

Fixed-page composition scale is absolute, uniform, and represented only by that existing profile/frame pair. Its `100%` extents are defined by contain-fitting the full frozen Composition at the locked drawable Page aspect; the centered version of that fit is the Reset reference, and an aspect mismatch therefore defines symmetric geometry-free padding rather than cropping. Every scale value derives Page Frame extents inversely from that stable fit instead of the prior draft, avoiding cumulative drift, and installs those extents around the frame's current center. Returning a panned frame to `100%` therefore preserves its current center. Reset is the sole operation that installs the centered reference position while returning scale to `100%`, without changing Page dimensions, margins, or the generation basis.

Completed Scene, Shading artwork, and Hidden-line geometry remain upstream of this cheap framing pass. Physical-tool Outline sources may reuse immutable completed Hidden-line geometry across profile and Page changes because their geometry contract is target-invariant; the active Tool width is reapplied as target-dependent styling before Page clipping, and the optional Page outline receives that same current physical width. Legacy Scene outlines make no target-invariance promise, so their authored artwork stroke widths remain part of strict cache identity and are preserved. This separation keeps pan and scale from invoking generation or expensive preparation while preventing a reused cache from carrying stale physical styling.

We rejected changing the Composition Frame whenever the user crops because aspect-sensitive Sketches would regenerate the artwork being preserved. We also rejected separate crop and padding models: both are the same Page Frame positioned respectively inside or beyond the Composition Frame, and separate representations would create avoidable state, UI, and export branches.

Studio therefore stores committed framing as one discriminated edit-history axis: either `unframed`, or `framed` with both the Page Frame and the generation aspect frozen from the original profile. Apply atomically commits that pair with the derived profile in ordinary mode or the exact locked profile in fixed-page mode. Re-editing continues from the committed frame while retaining the frozen aspect; Cancel remains outside edit history, while Apply and Reset are complete undoable commits. Ordinary Reset derives the identity full-Composition profile at the represented physical scale and returns to `unframed`; fixed-page Reset commits the centered contain-fit frame while retaining the locked profile. This makes invalid frame-without-basis states unrepresentable and lets Undo/Redo restore frame, generation basis, and physical profile together.

Fixed-page scaling adds no persisted composition transform and no migration envelope. The existing framed Preset/reproduction v3 record remains authoritative: generation aspect reproduces the frozen Composition, while the exact Output Profile and Page Frame reproduce scale, placement, dimensions, and insets. Composition scale is derived from those fields when the editor opens rather than serialized separately.
